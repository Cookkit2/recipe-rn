💡 What
Added missing `accessibilityRole`, `accessibilityLabel`, and `accessibilityState` props to the `AnimatedPressable` instances within the shared `GroupButton` components in both `SegmentedButtons.tsx` and `GridButtons.tsx`.

🎯 Why
Currently, the shared grid and segmented button components use a raw `Pressable` (via `AnimatedPressable`) without any accessibility tags. This means screen readers cannot announce their interactive nature (as a "button"), identify their label properly, or state whether they are currently selected. Adding these attributes significantly improves screen reader navigation across multiple screens that utilize these components.

📸 Before/After
No visual changes.

♿ Accessibility
- Added `accessibilityRole="button"` to allow screen readers to correctly announce the interaction type.
- Added `accessibilityLabel={item.label}` so screen readers read the button's label text, especially important if the icon rendering differs.
- Added `accessibilityState={{ selected: !!selected }}` so screen readers can announce whether the button is currently in the active/selected state.
